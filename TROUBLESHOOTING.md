# 문제 해결 가이드

## 마이크 접근 문제

### 1. 브라우저 개발자 도구로 확인

1. **F12** 키를 누르거나 **우클릭 > 검사** 클릭
2. **Console** 탭 클릭
3. 페이지 새로고침 (F5)
4. "시작" 버튼 클릭
5. 콘솔에 나타나는 메시지 확인

### 2. 콘솔 메시지 확인

**정상적인 경우:**
```
페이지 로드 완료
시작 버튼 클릭됨
AudioContext 초기화 시작...
AudioContext 초기화 성공: running
마이크 설정 시작...
getUserMedia 호출 중...
```

**에러가 있는 경우:**
- 에러 메시지를 확인하고 아래 해결 방법 참고

### 3. Chrome 브라우저 설정 확인

#### 마이크 권한 확인
1. 주소창 왼쪽의 **자물쇠 아이콘** 클릭
2. "사이트 설정" 클릭
3. "마이크" 항목이 "허용"인지 확인
4. "차단"이면 "허용"으로 변경
5. 페이지 새로고침

#### 전역 마이크 설정
1. Chrome 설정 열기 (chrome://settings/)
2. "개인정보 및 보안" > "사이트 설정"
3. "마이크" 클릭
4. "사이트에서 마이크 사용 전에 권한 요청" 체크 확인

### 4. Windows 시스템 마이크 권한

1. **Windows 설정** 열기 (Win + I)
2. **개인 정보** > **마이크**
3. **"앱이 마이크에 액세스하도록 허용"** 활성화
4. **"마이크에 액세스할 수 있는 앱 선택"**에서 Chrome 확인

### 5. HTTPS vs HTTP

- **localhost**는 HTTP에서도 마이크 접근 가능
- 다른 도메인에서는 **HTTPS** 필수

### 6. 다른 프로그램 점유 확인

다음 프로그램들이 마이크를 사용 중인지 확인:
- Zoom
- Microsoft Teams
- Discord
- Skype
- 기타 화상 회의 프로그램

사용하지 않는 프로그램을 종료하고 다시 시도

### 7. 브라우저 재시작

모든 설정 변경 후:
1. Chrome 완전히 종료
2. 다시 실행
3. 페이지 접속 후 재시도

### 8. Chrome 버전 확인

오래된 Chrome 버전은 문제가 있을 수 있습니다:
1. 주소창에 `chrome://version/` 입력
2. 버전이 최신인지 확인
3. 업데이트 필요시 업데이트
